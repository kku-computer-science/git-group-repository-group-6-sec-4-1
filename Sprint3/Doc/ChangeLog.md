# Product Backlog Item #19

## User Story
**As an admin,** I want a dashboard of user activity logs, error logs, and important system information logs.

---

## Changelog

### 📅 [27/02/2025]
#### ✅ Added
- เพิ่มฟีเจอร์ **Filter Log** เพื่อให้สามารถแสดงข้อมูลตามเงื่อนไขที่กำหนด
- แยกประเภท Log ออกเป็น 3 ประเภท:
  - **User Activity Log** 📝 (บันทึกการกระทำของผู้ใช้)
  - **HTTP Error Log** ⚠️ (บันทึกข้อผิดพลาดที่เกิดจาก HTTP Request)
  - **System Error Log** 🛠️ (บันทึกข้อผิดพลาดของระบบ)

#### 🔄 Changed
- ปรับปรุง **UI** ให้แสดงผล Log ได้เข้าใจง่ายขึ้น 🎨
- แก้ไข **Bug** ที่ทำให้บาง Log ไม่สามารถบันทึกได้ ❌📜

---

### 📅 [13/03/2025]
#### ✅ Added
- สร้าง **Dashboard สำหรับ Admin** เพื่อสรุปข้อมูลจากไฟล์ Log ให้ดูง่ายขึ้น 📊
- เพิ่ม **ระบบแจ้งเตือนข้อความสำคัญ** เช่น:
  - การพยายามล็อกอินที่ผิดปกติ 🚨
  - การเรียกใช้ **Path เดิมซ้ำๆ** ซึ่งอาจเป็นพฤติกรรมของ Bot 🤖
- **ตารางแสดงยอดผู้ใช้ที่มี Activity สูงสุด** ช่วยให้ติดตามพฤติกรรมผิดปกติของ User ได้ 🔍
- เพิ่ม **Route Debug Error** เพื่อใช้ในการ Debug ระบบ 🐞

#### 🔄 Changed
- เปลี่ยนการแสดงผล **Log เดิม** จากที่สามารถ Filter ได้อย่างเดียว ➡️ เป็น **การสรุปข้อมูลที่อ่านง่ายขึ้น** 📈
- แก้ไข **Bug** ที่ทำให้มีการบันทึกข้อมูลซ้ำใน Log ⚡
- ปรับแต่ง **UI ให้สวยงามยิ่งขึ้น** 🎨✨
